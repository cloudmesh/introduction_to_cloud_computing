Excersises¶

her is the task i like you to do to reinforce the learning experience once you are done with the tutorial.

  1. Write a program that uses loops over both x and y coordinates while x is in 1,2,3,4,5 and y is in 5,4,3,2,1 and prints the x and y coordinate
  2. Write a program that sums up all values in x and y
  3. Write a program just like the first task but does not print values where x is equal to 2 and y is equal to 4
  4. Write a function that takes in a word and returns it in reverse order
  5. Provide a program that uses dicts
  6. Read up on classes and write a program that uses classes.
  7. We will create an icecream machine that produces icecream in with tiny flavor, medium flavor and large flavor.
  8. In addition the icecream cone will be wrapped into some paper that has an image on it. Images will be Penguin, Apple, Emperor, King
